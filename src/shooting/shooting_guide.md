# Siglen · test_run_v1 · Shooting Guide

**Objective**  
Quick system validation run for Siglen factory automation content.  
Target: ~20s vertical short video (9:16).

---

## 1. Establish Context · 工厂环境建立

**Purpose**  
建立专业、干净、有规模的生产环境印象。

**Shots Required**
- **Wide**：生产线全景，稳定、干净
- **Medium**：生产线局部或主要区域

**Notes**
- 不要 detail 打头
- 优先稳定镜头（static / slow pan / slide）

---

## 2. Show Capability · 自动化能力展示

**Purpose**  
展示 Siglen 的自动化生产能力与系统成熟度。

**Shots Required (顺序非常重要)**
1. **Wide**：自动化流水线整体运行
2. **Medium**：机械臂 / 核心单元工作
3. **Detail**：关键动作、精度、控制细节

**Notes**
- 这是一个“连续观察段”，镜头之间应逻辑连贯
- detail 只用于强调，不抢戏

---

## 3. Build Trust · 测试与质量可信度

**Purpose**  
让观众“相信”这套系统是可靠、可交付的。

**Shots Required**
- **Wide / Medium**：测试区域整体
- **Detail**：测试动作、组件、读数、质检细节

**Notes**
- 节奏可稍快，但避免突兀切换
- detail 不要连续出现

---

## 4. Brand Close · 品牌收尾（Hero）

**Purpose**  
品牌记忆点，适合结尾或 Logo / Slogan 叠加。

**Shots Required**
- **Hero**：厂房、空间、设备整体
- 推荐运镜：static / orbit / reveal

**Notes**
- 画面尽量干净
- 可无字幕，留给后期广告字设计
