Pool Fill Shooting Guide

Compact field guide for photographers and operators using Pool Fill Mode.

How to read a slot

scene = where you are shooting
content = what object / action / process should appear
coverage = how wide or tight the frame should be
move = how the camera should move

Example translations

  • tower / testing / detail / static → capture a stable detail that clearly proves testing is happening.
  • factory / automation / wide / static → show the full automation area clearly with a stable wide shot.
  • showroom / product / hero / orbit → capture a premium rotating hero-style product shot.

If a slot cannot be shot on site

  1. Check whether the scene or process really exists.
  2. If the exact slot is unavailable, look for an honest nearby substitute.
  3. If there is no real substitute, update the pool plan.
  4. Do not silently invent new token names on site.
Practical rule

The token is for system consistency. The explanation is for the photographer. If the token is correct but hard to understand, improve the guide. If the scene itself is wrong, improve the pool plan.
