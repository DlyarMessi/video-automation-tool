# Authorship Statement

This project started as workflow automation scripts written by the author for
personal work use.

It then evolved through steady iteration into a broader and more structured
repository.

The director-layer rules, workflow design, and core UI concepts/layout were
led and originally defined by the author.

AI assistance was used during development as support for implementation,
refactoring, and optimization tasks.

Core creative direction and product decisions remain human-led by the author.
